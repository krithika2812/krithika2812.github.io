# Airway Restaurant Premium Landing Page

## What’s included
- Single-page premium landing page (HTML/CSS/JS)
- Hero + Trust badges + floating CTA
- Venue matrix (grid dashboard)
- Event versatility tabs (corporate/social/amenities)
- Sticky lead capture + pricing estimator
- Gallery with lazy loading (`loading="lazy"` + IntersectionObserver)
- SEO semantics + basic JSON-LD schema
- WhatsApp integration hook (front-end `wa.me` link; replace phone number)

## Run locally
1. Open `index.html` in a browser.
2. (Optional) Use a static server for testing.

### Optional: VSCode Live Server
- Install **Live Server** extension
- Right-click `index.html` → **Open with Live Server**

## WhatsApp integration
In `script.js`, update:
- `BUSINESS_WA_NUMBER = '9000000000'`

Replace it with the real WhatsApp business number (without +91).

## Replace placeholders
- Replace JSON-LD fields (address/telephone/url) as needed
- Replace gallery images with actual banquet hall images
